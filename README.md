Developed a responsive e-commerce website inspired by Amazon using HTML and CSS. The project replicates the basic user interface of Amazon, including the homepage layout, navigation bar, product sections, search bar, banners, and footer design. Focused on creating a visually appealing and user-friendly interface while improving frontend development and responsive web design skills.

Key Features
Responsive homepage design
Navigation bar with search functionality UI
Product listing sections and banners
Footer with multiple informational links
Clean and user-friendly interface
Mobile-friendly layout using CSS styling
